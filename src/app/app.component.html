Create dynamic components using the below options provided by Angular:
  1. ComponentFactoryResolver
  2. ngComponentOutlet
Project Structure
  1. Dynamic Components: There are two dynamic components which inherit from BaseForm model:
    • ComponentFactoryResolverComponent (component-factory-resolver.component)
    • NgComponentOutletComponent (ng-component-outlet.component)
  2. App.module.ts: These dynamic components are registered with AppModule 
            ...
        entryComponents: [
        ComponentFactoryResolverComponent,
        NgComponentOutletComponent        
        ]
  3. BaseForm model is inherited by the dynamic components. It contains an 'ImageUrl' property that needs to filled by the calling/parent component.
  4. ProjectStructureModel has 'Title' property that needs to filled by the calling/parent component.
  5. ProjectStructureFactory is a factory class to inject data & singleton service in the dynamic components. This is used in Component1 & Component2.
  6. Component1 loads ComponentFactoryResolverComponent using ComponentFactoryResolver & createComponent method.
    The createComponent method returns an instance of the dynamic component.
  7. Component2 loads NgComponentOutletComponent using *ngComponentOutlet directive.
  8. ComponentInfoService class is an Observable class that is used to pass an instance of the dynamic component.
    This is used only in NgComponentOutletComponent. In there during the ngOnInit event, this service class passes its instance to the listener i.e. Component2 .
  9. Routing: 
            { path: 'ComponentFactoryResolver', component: Component1 },
        { path: 'ngComponentOutlet', component: Component2 }
